Janilla provides a full web solution stack
packed into a single lightweight library

Vanilla everywhere
Just plain Java for the backend and plain JavaScript, HTML and CSS for the frontend, all in their most modern form: there are no additional entry barriers.
Truly open-source
Reasonably few and short source files, simply organized to facilitate quick and comprehensive understanding: the learning curve is pretty approachable.
Problem solving driven
Getting your hands dirty with computing is the key to produce quality software in an efficient way: hence our library comes with a rich set of sample applications.
Backed by Computer science
Core library components have been implemented leveraging efficient data structures, and we are committed to further advance the state-of-the-art.
Developer friendly
Few concepts to learn a dependency-free and compact architecture based on mature and well-established technologies, to offer a smooth productivity experience.
Everything under control
Only one thin library separates the application from the server and the browser, and you have the possibility to access, debug and improve any component in any layer.
